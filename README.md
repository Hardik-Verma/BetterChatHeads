**Better Chat Heads**

**Overview**
Better Chat Heads is a client-side Fabric mod that shows player heads next to chat messages in multiplayer.
It focuses on cleaner sender matching, smarter skin recovery, better support for rank-heavy server chat, and a polished in-game config screen.

**Main Features**
- Player heads beside recognized player chat lines
- **Smart Skin Sync+** for retrying missing skins
- **Live Skin Swap** for smooth fallback-to-real-skin updates
- **Per-Server Profiles** for separate settings on different servers
- **Server Learning Cache** for remembering working chat separators and prefixes
- Optional own-message head rendering
- Overlay layer support for hats, hair, and second-layer skin details
- Compact Chat Mode for tighter multiplayer chat spacing
- Adjustable chat animation and head intro behavior
- Built-in config screen with instant saving
- Optional Mod Menu support

**Why Use It**
- Makes busy multiplayer chat easier to scan
- Keeps the mod client-side only
- Works better on servers with ranks, prefixes, and custom separators
- Looks cleaner than plain text-only chat while staying lightweight

**Version**
**v2.0.0**

**Supported Setup**
- Minecraft **1.21.11**
- Fabric Loader **0.19.2**
- Fabric API **0.141.3+1.21.11**
- Java **21**

**General Compatibility**
This release line is designed for modern **1.21.x** Fabric environments, but separate branch builds may exist for older minor versions like **1.21.10**, **1.21.8**, **1.21.5**, or **1.21.4**.

**Config Access**
- Press **B** by default to open the Better Chat Heads config screen
- Changes apply instantly
- Extra keybinds for toggling heads, overlay layer, and head size can be configured in controls

**Notes**
- **Client-side only**
- Servers do **not** need to install the mod
- System messages and unresolved lines stay headless on purpose
- If a real skin cannot be resolved, the mod can fall back to a default head

**Short Upload Description**
Better Chat Heads **v2.0.0** adds cleaner multiplayer head rendering, smoother skin syncing, per-server settings, smarter server parsing, and a more polished in-game UI.
