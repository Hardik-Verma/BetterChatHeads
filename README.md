# BetterChatHeads

BetterChatHeads is a lightweight client-side Minecraft mod that adds player heads next to chat messages, making multiplayer chat clearer, faster to read, and more visually intuitive.

## ✨ Features
- 🗨️ Player heads in chat for better readability
- 📋 Works in multiplayer and singleplayer
- 🎨 Proper skin rendering (supports outer layers, no bald heads)
- 🔁 Supports both premium and cracked servers
- ⚡ Lightweight and performance-friendly
- 🧩 Clean, modern chat formatting

## 🚀 Why use BetterChatHeads?
Vanilla Minecraft chat is text-only and hard to scan in busy servers.  
BetterChatHeads improves chat visibility by adding visual identity to every message.

Perfect for:
- SMP servers  
- PvP servers  
- Active multiplayer chats  

## ⚙️ Compatibility
- Minecraft Fabric (client-side only)
- No server installation required
- Works on any server

## 📥 Installation
1. Install Fabric Loader  
2. Install Fabric API  
3. Place the mod `.jar` in your `mods` folder  
4. Launch Minecraft
